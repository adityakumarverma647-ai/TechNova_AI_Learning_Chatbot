# AI Learning Chatbot Using Rule-Based Logic

## Project Overview

This project is a console-based chatbot developed using Python. The chatbot uses rule-based logic and pattern matching to respond to user queries related to Artificial Intelligence, Machine Learning, Python, and Data Science.

## Objectives

* Build a chatbot using Python.
* Implement rule-based responses using if-else statements.
* Handle multiple user intents.
* Provide a fallback response for unknown inputs.

## Features

* Greeting and farewell messages.
* Personalized user interaction.
* Supports more than 10 user intents.
* Displays current date and time.
* Counts user questions.
* Handles unknown inputs gracefully.

## Technologies Used

* Python
* VS Code

## Future Enhancements

* Natural Language Processing (NLP)
* Voice Assistant Integration
* Graphical User Interface (GUI)
* AI-powered responses

## Conclusion

This project demonstrates the fundamentals of chatbot development using rule-based logic. It serves as a beginner-friendly introduction to Artificial Intelligence applications.
